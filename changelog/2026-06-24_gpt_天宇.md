## 失败原因

核心错误是 Parcel 在 Windows 下处理路径时，将绝对路径错误拼接成类似：

```text
C:\mw\mattermost-master\webapp\platform\shared\C:\mw\mattermost-master\webapp\platform\shared\src\context
```

导致：

```text
ENOENT: no such file or directory, mkdir ...
```

因为 `@mattermost/shared` 没有成功生成 `dist`，后续 `channels` 前端编译继续出现大量模块缺失错误，例如：

```text
Package path ./utils/user_agent is exported from package @mattermost/shared,
but no valid target file was found
```

最终 webpack 结果：

```text
502 errors
49 warnings
```
